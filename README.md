# Qun Chen Academic Website

Personal academic homepage for Qun Chen, PhD.

This site is built with Jekyll using an Academic Pages / Minimal Mistakes style structure inspired by the referenced academic homepage repository.

## Structure

- `_config.yml`: site metadata and author sidebar.
- `_pages/about.md`: homepage content.
- `_pages/publications.md`: publications index.
- `_publications/`: individual publication records.
- `cv.html`: standalone CV page.
- `files/CV.pdf`: downloadable CV.

## Local Preview

Install the GitHub Pages/Jekyll dependencies, then run:

```bash
bundle install
bundle exec jekyll serve
```

For deployment, push this repository to `qun37018-sudo/qun37018-sudo.github.io` and enable GitHub Pages.
